وصف مشروع كشف التصيد الاحتيالي (Phishing Detector) This Python project is a simple yet effective phishing detector tool designed to analyze URLs and determine their legitimacy. It checks for suspicious top-level domains, keywords commonly used in phishing links, and the use of URL shorteners to alert users about potentially harmful links.

Features:

Detects suspicious domain extensions often associated with phishing.

Searches for common phishing-related keywords within URLs.

Identifies shortened URLs as potential threats.

Provides clear warnings and reasons for flagging links.

Use case: This tool helps users, cybersecurity enthusiasts, and professionals quickly identify potentially fraudulent URLs to avoid phishing attacks and enhance digital security awareness.# Phishing-Detector
